# Prism | Minimalist Workspace

**Prism** is a premium, minimalist productivity dashboard designed to help you focus and stay inspired. It's built with a stunning glassmorphic UI and animated aurora backdrop.

## Core Features
1. **Focus Center**: A sleek, high-end Pomodoro timer with a circular progress indicator.
2. **Zen Tasks**: A minimalist task manager for focusing on the essentials.
3. **Daily Wisdom**: Rotating curated quotes from history's most creative minds.
4. **Persistence**: Your tasks are automatically saved to your browser's local storage.
5. **Glassmorphic UI**: A modern interface with vibrant, animated background gradients.

## Technology Stack
- **HTML5**: Semantic structure and modern meta tags for SEO.
- **Vanilla CSS3**: Custom properties, backdrop-filters, and keyframe animations.
- **Vanilla JavaScript**: Lightweight, performant logic for all interactions.

## Getting Started
To run Prism locally:
1.  Open the `index.html` file in any modern web browser.
2.  Start your focus session and get productive!

## Design Aesthetics
Prism follows a "dark-mode-first" design philosophy, utilizing HSL color palettes and smooth gradients to create a serene environment for work and creativity.
